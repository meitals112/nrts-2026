Category: Graduate
Type: Interactive Installation
Year: 2026
Artist: Meital Geva

About
NRTS is a speculative interactive installation staged as an accredited certification clinic. The work imagines emotional neutrality as a professional requirement: measured, trained, certified, and reported.
The installation is a functioning local system — not a screen simulation. It runs through a browser interface connected to a Python / FastAPI backend, using webcam and microphone input, MediaPipe facial tracking, voice analysis, and VAD-based scoring. At the end of each session, a Brother QL-700 label printer outputs a physical certification label.
